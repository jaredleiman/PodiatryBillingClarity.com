CPT Code 11056: Corn and Callus Removal (2-4 Lesions) Billing Guide

By PodiatryBillingClarity · April 2026 · 7 min read

CPT 11056 covers the paring or cutting of 2 to 4 benign hyperkeratotic lesions — commonly known as corns and calluses. Along with 11055 (single lesion) and 11057 (5+ lesions), this code series represents a core service in podiatric practice and a frequent target of Medicare audits.

What 11056 Covers

11056 is used when you pare, trim, or reduce 2 to 4 separate benign hyperkeratotic lesions (corns, calluses, or keratomas) during a single encounter. The procedure involves skilled reduction of thickened skin using a scalpel, blade, or other instrument. Each lesion must be distinct and separately identifiable — a single large callus covering a broad area is one lesion, not multiple.

11055 vs. 11056 vs. 11057

11055 — Single lesion. 11056 — 2 to 4 lesions. 11057 — 5 or more lesions. These codes are mutually exclusive — bill only one code per encounter based on the total lesion count. If you pare 4 calluses, bill 11056. If you pare 5, bill 11057. The count must be documented precisely.

Medicare Coverage

Callus and corn paring is classified as routine foot care under Medicare. Like nail debridement, it is only covered when the patient has a documented Class Finding — a systemic condition that makes professional care medically necessary to avoid hazard. The same Q modifiers (Q7, Q8, Q9) apply. Without a Class Finding, Medicare denies the claim. The patient can be billed directly for non-covered routine care only if an ABN was signed before the service.

Documentation Requirements

Each 11056 claim must include the exact number of lesions pared (must be 2-4), the location of each lesion (e.g., "plantar 1st metatarsal head left foot, plantar 5th metatarsal head left foot, dorsal 5th toe PIP joint right foot — 3 lesions total"), description of each lesion (size, depth, character), the technique used, the Class Finding documentation for Medicare patients, and the Q modifier on Medicare claims.

Common audit finding: Notes that say "calluses pared bilaterally" without specifying the number and location of each individual lesion. An auditor cannot determine whether 11055, 11056, or 11057 is appropriate without a documented lesion count.

Billing 11056 With Other Codes

11056 can be billed on the same day as nail debridement (11720/11721) — these are different services addressing different structures. 11056 can also be billed with an E/M code (with modifier 25) if a separately identifiable evaluation and management service was performed. 11056 should not be billed for debridement of a lesion that is part of a wound being treated under wound debridement codes (11042-11047).

Common Billing Errors

  • Counting one large callus as multiple lesions — A broad plantar callus is one lesion regardless of how large it is
  • Not specifying lesion locations — Each lesion must be individually identified in the note
  • Billing 11056 and 11055 together — These codes are mutually exclusive; use the one that matches the total count
  • Missing Class Finding for Medicare — Automatic denial without it

The Bottom Line

11056 is a bread-and-butter podiatry code that's straightforward to bill when documented correctly. Count every lesion, document every location, support with a Class Finding for Medicare, and use the right Q modifier. The practices that avoid audit problems with callus codes are the ones that treat lesion counting as a clinical documentation standard, not an afterthought.
