Podiatry CPT Codes: Complete Billing Reference

By PodiatryBillingClarity · April 2026 · 10 min read

Podiatry billing uses a specific set of CPT codes spanning routine foot care, surgical procedures, evaluation and management, and durable medical equipment. This guide covers every commonly used podiatry code with definitions, Medicare coverage rules, and billing notes.

All routine foot care codes require a Class Finding for Medicare coverage — a documented systemic condition (diabetes, PVD, neuropathy) that makes professional foot care medically necessary. Without the Class Finding and appropriate Q modifier, these claims will be denied.

Nail surgical codes are medical procedures, not routine foot care — they do not require a Class Finding. They do require standard medical necessity documentation: the condition being treated, the clinical findings, and the rationale for surgical intervention over conservative treatment.

Evaluation and Management Codes

Podiatrists use standard E/M codes (99202-99215) for office visits. The level of E/M code is determined by medical decision-making complexity, as updated in the 2021 E/M guidelines. Key points for podiatrists: you can bill an E/M code on the same day as a procedure if the E/M represents a separately identifiable service (use modifier 25), do not bill an E/M code for the work included in a procedure code (the surgical package includes the pre-procedure assessment), and Medicare pays podiatrists the same E/M rates as other physicians.

Radiology Codes

Podiatrists frequently order and interpret foot and ankle X-rays. Common codes include 73620 (foot X-ray, 2 views), 73630 (foot X-ray, complete, minimum 3 views), and 73600 (ankle X-ray, 2 views). Bill the professional component (modifier 26) if you interpret the X-ray but another facility owns the equipment. Bill the global code (no modifier) if you own the X-ray equipment and perform the interpretation.

Billing tip: When billing an E/M code with a minor procedure on the same day, append modifier 25 to the E/M code to indicate the evaluation was a separately identifiable service from the procedure. The E/M documentation must clearly support a separate medical decision-making process beyond what's included in the procedure's global period.

The Bottom Line

Podiatry billing spans routine foot care, surgical procedures, and medical evaluation — each with distinct coverage rules and documentation requirements. The most important distinctions to master are routine vs. medical (Class Finding required for routine), the nail count threshold (11720 vs. 11721), and proper use of modifier 25 for same-day E/M billing. Accurate code selection combined with thorough documentation is the foundation of compliant, well-reimbursed podiatry billing.
