CPT Code 11055: Paring of Single Benign Lesion Billing Guide

By PodiatryBillingClarity · April 2026 · 6 min read

CPT 11055 covers the paring or cutting of a single benign hyperkeratotic lesion — one corn, one callus, or one keratoma. It is the lowest-level code in the 11055-11057 lesion paring series and is used when only one distinct lesion is treated during the encounter.

When to Bill 11055

Bill 11055 when you pare exactly one benign hyperkeratotic lesion. Common scenarios include a single plantar callus under a metatarsal head, a single interdigital corn (heloma molle), a single dorsal corn over a PIP joint, or a single seed corn on the heel. The lesion must be benign. If there is any suspicion of malignancy, the specimen should be sent for pathology and a different code (biopsy or excision) is appropriate.

Documentation Requirements

For 11055, your note must include the location of the lesion (specific anatomical site), the type and description (size, depth, character), the technique used (blade, scalpel, curette), the Class Finding documentation for Medicare patients, and the Q modifier on Medicare claims. A compliant note: "Pared single plantar callus, 2cm diameter, plantar surface 3rd metatarsal head, left foot. Reduced with #15 blade to smooth contour. Patient is Type 2 DM with documented peripheral neuropathy (Q7). Professional care required due to loss of protective sensation — self-care would risk skin breakdown."

Lesion counting tip: A single broad callus is one lesion regardless of size. Do not count bilateral occurrences of the same callus pattern as multiple lesions — each individual, distinct callus formation is one lesion. Bilateral plantar calluses under the 1st metatarsal heads are two lesions (one per foot), billed as 11056.

11055 vs. 11056 vs. 11057

The only difference between these codes is the lesion count. 11055 is for 1 lesion, 11056 for 2-4, and 11057 for 5 or more. They cannot be billed together — use the single code that matches your total count. If you pare one callus and then notice a second one that also needs treatment, the visit becomes 11056, not 11055 + 11055.

Billing 11055 With Other Codes

11055 can be billed on the same day as nail debridement (11720/11721) since these address different structures. 11055 can be billed with an E/M code (modifier 25) when a separately identifiable evaluation is performed. 11055 should not be billed alongside wound debridement codes for the same lesion site.

The Bottom Line

11055 is the simplest code in the callus paring series — one lesion, well-documented, with a Class Finding for Medicare. The key is accurate lesion counting and location documentation. When the count goes above one, move to 11056 or 11057. Never bill two units of 11055 — that's a coding error.
